Electric Vehicle Application | MEVS

  • Category: Windows Application
  • Built With: C#, MySQL, XAML
  • Project date: May, 2023
Source Code

A C# application based on a 3-tier software architecture, designed to support user login and registration across multiple roles, including EV owners, charging station owners, and EV service providers. This platform functions as an advanced EV data bank, streamlining the subscription process for EV owners. Users can modify their subscriptions to service providers, monitor their engagements, and generate detailed reports through an intuitive dashboard, ensuring a seamless and efficient experience in managing EV-related services.
